# Forest Fires EDA Project

## Overview

This project performs Exploratory Data Analysis (EDA) on the Forest Fires dataset to understand the factors affecting forest fire occurrences and burned area. The analysis includes data cleaning, preprocessing, visualization, feature engineering, encoding, scaling, and dataset preparation for machine learning.

## Dataset Features

* X, Y : Spatial coordinates
* Month : Month of the year
* Day : Day of the week
* FFMC : Fine Fuel Moisture Code
* DMC : Duff Moisture Code
* DC : Drought Code
* ISI : Initial Spread Index
* Temp : Temperature (°C)
* RH : Relative Humidity (%)
* Wind : Wind Speed
* Rain : Rainfall
* Area : Burned Area (Target Variable)

## Project Workflow

1. Data Loading and Inspection
2. Exploratory Data Analysis (EDA)
3. Statistical Summary
4. Distribution Analysis using Histograms
5. Correlation Analysis
6. Month and Day Encoding
7. Train-Test Split
8. Scatter Plot Visualization of Fire Locations
9. Fire Density Analysis
10. Feature Scaling using StandardScaler
11. Label Encoding
12. Custom Transformers and Pipelines
13. Dataset Preparation for Machine Learning

## Key Insights

* Temperature and humidity significantly influence fire occurrence.
* Most fires are concentrated in specific forest locations.
* Burned area distribution is highly skewed.
* Feature scaling improves machine learning readiness.
* Spatial coordinates help identify fire-prone regions.
